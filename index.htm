What is Absurd Industries?

We're all about bending the rules and defying the norm. We're here to push boundaries, and, quite frankly, to make science and technology a whole lot more fun.

Current Projects

  • Lightsaber: Building a lightsaber using the open-source Brewboard and SaberOS
  • DJ Kit and Gaming Console: Assembling kits from CircuitMess
  • Petoi Cat/Dog: Creating an open-source four-legged robot
  • Tatooine Diorama: Crafting a Star Wars-themed diorama
  • LED Matrix: Constructing a wearable LED matrix
  • InMoov Hand: Working on a part of the open-source InMoov robot
  • Quad Bot Spider Walker: Building a Raspberry Pi Pico-powered robot
  • Firefly Lamp: 3D printed voronai lamp that simulates fireflies
