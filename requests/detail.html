Open Source Desktop CNC Machine

Description

We need a compact, affordable desktop CNC router that's fully open-source from the ground up. The machine should handle PCB milling, engraving, and light woodwork with reasonable precision for hobbyist and small-workshop use.

The key requirement is that everything must be open: Grbl-compatible firmware, KiCad-designed control board, FreeCAD mechanical design files, and a full BOM sourced from commonly available parts. No proprietary black boxes.

Why this matters:

  • Existing affordable CNC machines use proprietary firmware that can't be modified
  • Open-source alternatives (like MPCNC) require significant build knowledge
  • Makers in developing countries need locally-repairable machines
  • PCB milling reduces dependence on overseas PCB fab for prototyping

An optional enclosure design would be a bonus for dust collection and noise reduction. The goal is a machine that a competent maker can build in a weekend from the provided files and a reasonable BOM.
